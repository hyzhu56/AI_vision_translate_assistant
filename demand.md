请帮我用 Python 开发一个名为 "AI_vision_translate_assistant" 的 Windows 桌面悬浮工具。

【要求】

操作系统： Windows

AI 接口： 使用 openai 库对接 Moonshot (Kimi 2.5) 的兼容 API。

【核心功能流程】

全局唤醒： 程序后台运行。用户按下全局快捷键 Ctrl + CapsLock。

屏幕冻结与框选： 触发快捷键后，立刻截取当前全屏，并弹出一个置顶的、全屏无边框的 PyQt6 窗口显示该截图。屏幕呈现半透明黑色遮罩，鼠标光标变为十字。用户通过鼠标拖拽画出一个矩形框选区域。

内存截取： 鼠标松开后，全屏遮罩消失。程序将用户框选区域的图像在内存中转换为 Base64 格式（不要在本地生成图片文件）。

智能请求： 将 Base64 图片发送给 Kimi API (作为多模态消息)。

System Prompt 设定为： "你是一个智能视觉助手。请分析用户提供的图片。如果图片主体是普通英文文本，请提供信达雅的中文翻译。如果图片主体是编程代码，请提供深度的代码解析（包含编程逻辑、语法解析、关键函数分析）。使用 Markdown 格式输出。"

悬浮面板展示： 在屏幕右下角或选区附近，弹出一个现代化、美观的悬浮窗口，流式打印 Kimi 的返回结果。

【UI 与交互细节要求】

悬浮面板 UI： 深色模式，圆角无边框设计。具备一定的半透明效果 (Acrylic 风格更佳)。使用 QTextBrowser 渲染 Markdown 结果。

图钉功能： 面板右上角必须有一个“图钉”图标按钮。

默认状态（未固定）：当面板失去焦点（用户点击了屏幕其他地方），面板自动隐藏/关闭。

固定状态（已固定）：点击图钉后，窗口强制置顶，失去焦点也不会消失，方便用户对照代码学习。再次点击取消固定。
